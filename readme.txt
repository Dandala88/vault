Vault

A game that has you robbing a bank vault where each goin you pickup makes you heavier and so it makes you slower.

Use arrowkeys to move.

Hold d to drop coins.  While holding d you can press space the number of coins you would like to drop.

Move away from dropped coins while holding d and when you pass over the dropped coins without holding d you will pick them back up.

This game is being developed for my ludum dare 40 prep.

Uses the love2D framework
